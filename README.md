# E-Commerce Database (SQL Server)

A SQL Server database project for an e-commerce system, built as part of my SQL learning journey.

## Project Structure

* `01_ecommerce_schema.sql` – Database schema (tables, primary keys, foreign keys, and constraints)
* `02_ecommerce_sample_data.sql` – Sample data for testing
* `03_update_examples.sql` – UPDATE statement examples and practice
* `04_delete_concept.sql` – DELETE statement concepts and practice
* `05_alter_truncate_drop.sql` – ALTER, TRUNCATE, and DROP statement examples
* `06_data_querying.sql` – Data querying concepts and practice (SELECT, DISTINCT, WHERE, Comparison Operators, Logical Operators (AND, OR, NOT), BETWEEN, IN,  LIKE, ORDER BY, TOP, OFFSET, FETCH)

## Status

🚧 Work in progress — new SQL files, concepts, and features will be added as I continue learning SQL Server.
